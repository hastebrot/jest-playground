## jest-playground

Setup environment.

```
❯ node --version
v17.6.0

❯ yarn --version
1.22.17
```

Install dependencies.

```
❯ yarn install
```

Run test suite.

```
❯ yarn jest --silent --watchAll
```

Run code formatter.

```
❯ yarn prettier . --write
```
